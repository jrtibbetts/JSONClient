REST methods
  • get(path:headers:parameters:)

    Get the JSON data at the specified path (relative to the baseUrl) and decode it to the expected type.

    Throws

    JSONError.invalidUrl if path contains illegal characters, JSONError.nilData if path is valid, but has no content that can be parsed, JSONError.parseFailed if the JSON at path isn’t valid for the expected type.

    Parameters

    path

    The path of the URL, relative to the baseUrl, or, the absolute URL if baseUrl is nil.
    headers

    Custom header values to use in the request.
    parameters

    Parameter name/value pairs to be appended to the URL.
