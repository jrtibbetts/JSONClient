OAuth2JSONClient

open class OAuth2JSONClient : AuthorizedJSONClient

An AuthorizedJSONClient implementation that uses OAuth 2 for authentication and authorization.

  • init(consumerKey:consumerSecret:authorizeUrl:accessTokenUrl:baseUrl:)

    Initialize the client with the app’s hashes on the server, as well as the server’s various OAuth-related URLs.

    public init(consumerKey: String,
            consumerSecret: String,
            authorizeUrl: String,
            accessTokenUrl: String,
            baseUrl: URL? = nil)

    Parameters

    consumerKey

    The hash that identifies the app that this client communicates with. It’s also called a client key.
    consumerSecret

    The password hash for the server app. This value should not be included in publicly-visible code.
    authorizeUrl

    The URL of the app’s sign-in page.
    accessTokenUrl

    The URL where the OAuth client can exchange a request token for an access token.
    baseUrl

    The service’s root-level URL. All relative paths that are passed to the various REST functions will be resolved against this URL. If it’s nil, then all paths must be absolute URL strings.
  • authorize(presentingViewController:callbackUrlString:scope:)

    Launch the service’s sign-in page in a modal Safari web view. After the user has successfully authenticated, the web page will be redirected to the callback URL, which is unique to the client application.

    Declaration

    Swift

    open func authorize(presentingViewController: UIViewController,
                    callbackUrlString: String,
                    scope: String = "") -> Promise<OAuthSwiftCredential>

    Parameters

    presentingViewController

    The view controller over which the Safari view controller will be presented modally.
    callbackUrlString

    The URL that the web view will load after authentication succeeds.
    scope

    The level of access that the client is requesting. See the server’s documentation for what scopes are supported. The default value is an empty string.

    Return Value

    A Promise containing whatever type of data is sent back by the server after authentication succeeds.
